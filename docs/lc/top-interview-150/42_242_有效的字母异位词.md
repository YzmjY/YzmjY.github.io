# 242. 有效的字母异位词

## 题目

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的 字母异位词。

 

**示例 1:**

> 输入: s = "anagram", t = "nagaram"  
> 输出: true


**示例 2:**

> 输入: s = "rat", t = "car"  
> 输出: false
 

## 题解

## 代码

=== "Go"

    ```go
    func isAnagram(s string, t string) bool {
        var dict = make(map[byte]int)

        if len(s) != len(t) {
            return false
        }

        n := len(s)
        for i := 0; i < n; i++ {
            dict[s[i]]++
            dict[t[i]]--
        }

        for _, v := range dict {
            if v != 0 {
                return false
            }
        }
        return true
    }
    ```
